1. Руководство пользователя

    1) Убедиться в наличии Python 3
        -Убедиться в наличии selenium для Python
        -Убедиться в наличии rfeed для Python
            !В слуаче отсутствия установить

    2) Убедиться в наличии Chrome или Firefox
            !В случае отсутствия установить

    3) Указать в файле settings.txt используемый браузер напротив поля browser:

    4) Запустить из командной строки файл main.py


2. Руководство администратора

    Скрипт состоит из четырёх модулей:
        1) Browser - класс отвечающий за инициализацию скриптов
        2) news_getting - класс, методы которого парсят информацию с сайтов транспортных вузов
        3) feedgenerator - функции, отвечающие за генерацию XML таблиц в формате RSS
        4) main - main

    Файл settings.txt
        -Поле browser отвечает за тип инициализированного браузера
        -Поле url_list отвечает за ссылки на главные страницы сайтов

    chromedriver.exe и geckodriver.exe
        Драйвера для работы selenium с браузерами соответственно: Chrome и Firefox

3. Требования к среде эксплуатации

    -Установленный Python 3
    -Установленные библиотеки seleniun и rfeed
    -Установлен хотя бы один из браузеров: (Chrome,Firefox)
    -Операционная система Windows 7 и выше.
